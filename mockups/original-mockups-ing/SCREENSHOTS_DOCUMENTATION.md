## 8. Workflow Connections Summary

This section provides a high-level overview of how the different workflows connect and the navigation patterns within the ING App.

### Main Navigation Structure:

```
┌─────────────────────────────────────────────────────────────────┐
│                        ING Banking App                           │
├─────────────────────────────────────────────────────────────────┤
│  Bottom Navigation:                                              │
│  [Übersicht] [Karten] [Produkte] [Postbox] [Mehr]              │
└─────────────────────────────────────────────────────────────────┘
```

### Workflow Entry Points:

| Workflow | Entry Point | Navigation Path |
|----------|-------------|-----------------|
| App Setup | Fresh Install | Launch App → Welcome → Setup Wizard |
| Learn App | First Login / Settings | Settings → App Tour, OR Automatic after setup |
| Card Limit | Dashboard / Cards Tab | Cards → Select Card → Limit ändern |
| Transfer | Dashboard Quick Actions | Dashboard → Überweisen button |
| Access Management | More Menu | Mehr → Zugang verwalten |
| Depot Transfer | Products Tab | Produkte → Depot → Depot übertragen |

### Cross-Workflow Connections:

1. **App Setup → Learn App:** After completing initial setup, users are typically guided through the "Learn Your App" tour.

2. **Dashboard → All Workflows:** The main dashboard serves as the central hub, with quick actions for transfers and navigation to all other sections.

3. **Cards Section ↔ Dashboard:** Card information and limits can be accessed from both the dedicated Cards tab and from card widgets on the dashboard.

4. **More Menu → Access/Security:** Security and access management features are accessed through the More menu, which also connects to settings, help, and profile.

5. **Products → Investment Workflows:** Investment-related workflows like depot transfer are accessed through the Products section.

### Common UI Patterns:

- **Orange CTA Buttons:** Primary actions use orange background
- **Back Navigation:** "Abbrechen" (Cancel) in top-left
- **Screen Titles:** Centered in the navigation bar
- **PIN Entry:** Consistent 5-digit pattern with circles
- **Success States:** Green checkmarks with confirmation messages
- **Loading States:** Orange spinner/progress indicators

### Authorization Flow (Common to Many Workflows):

```
Action Initiated → Review Screen → Tap Confirm 
→ PIN Entry OR Biometric → Processing → Success/Failure → Return
```

This authorization pattern is used for:
- Money transfers
- Card limit changes
- Security setting changes
- Investment operations

Screenshots follow the pattern:
```

```

- Video: "ING App einrichten" (Setting Up the ING App)
- Timestamp: 0 minutes, 16 seconds into the video
- Type: screenshot
